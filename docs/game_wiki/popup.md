# 弹窗知识库 (Popup)

> 所有可能出现的弹窗/弹层。**RecoveryManager 处理"卡住恢复"时,先来这里查**。

---

## 1. 任务名称

弹窗知识库

## 2. 任务作用

- 整理游戏中所有可能的弹窗类型
- 每个弹窗:触发条件 / 表现 / 关闭方式 / 模板价值
- 帮 RecoveryManager 快速识别并恢复

## 3. 入口位置

> 弹窗**没有固定入口**,会从任意页面/任务/事件中触发。

---

## 4. 弹窗分类

按"是否全屏遮挡"分两类:
- **全屏弹窗**(必须先关才能操作主页) — `dismiss_x` 必扫
- **子弹窗 / 半屏弹窗**(不挡主页功能) — `dismiss_x` 选扫

按"出现频率"分:
- **高频**(每天):邮件奖励弹窗 / 教学弹窗 / 战区聊天 / 段位赛奖励
- **中频**(每周):公告 / 跨服要塞战倒计时 / 周末活动
- **低频**(每月):更新提示 / 维护公告 / 充值弹窗

## 5. 全屏弹窗清单

### 5.1 教学剧情弹窗 (高频)

| 项 | 内容 |
|---|---|
| 触发 | 教学任务触发(每次升级/新功能解锁) |
| 表现 | 屏幕中央剧情对话 + 角色立绘 + "下一步"按钮 |
| 位置 | 中央(占 80% 屏宽) |
| 颜色 | 半透明黑背景 + 白色对话框 |
| 关闭方式 | **点中央继续**(不自动关) |
| 模板价值 | ⭐ 高(教学剧情每账号固定出现 1-2 次) |
| 自动化处理 | RecoveryManager 检测到教学剧情 → 点中央 → 等 1-2s → 再点 → 直到剧情结束 |
| 已知陷阱 | 点中央不会**直接**关弹窗;会"剧情继续";需多次点 |

**新账号典型教学剧情**:
- 第一次进游戏:剧情介绍
- 解锁邮件:引导进邮件
- 解锁奖励:引导进奖励中心
- 解锁组织:引导加组织
- 解锁积分赛:引导进积分赛
- 解锁忍界指引:引导进忍界指引

### 5.2 顶部全屏活动弹窗 (中频)

| 项 | 内容 |
|---|---|
| 触发 | 限时活动开启 / 登录奖励 / 推送 |
| 表现 | 全屏覆盖 + 活动 Banner + "立即前往"或"我知道了"按钮 |
| 位置 | 全屏 |
| 颜色 | 主题色(随活动变) |
| 关闭方式 | **顶部 X 关闭** `(80, 60)` |
| 模板价值 | ⭐ 高(但 Banner 内容经常变) |
| 自动化处理 | RecoveryManager 检测到全屏活动 → 点 (80, 60) 关闭 |

**已知全屏活动类型**:
- "X 天后开启新活动" 倒计时弹窗
- 登录奖励 7 天弹窗
- "首充礼包" 充值弹窗
- "限时折扣" 商城弹窗
- "新忍者上线" 招募弹窗

### 5.3 段位赛奖励弹窗 (中频)

| 项 | 内容 |
|---|---|
| 触发 | 段位赛结束后(每天 21:00 结算) |
| 表现 | 段位图标 + 排名 + 奖励 + "领取"按钮 |
| 位置 | 屏幕中央 |
| 颜色 | 金/银/铜(按段位) |
| 关闭方式 | **"领取"按钮** `(960, 750)` |
| 模板价值 | ⭐ 中(段位图标可能变) |
| 自动化处理 | 段位赛结束 → 自动弹窗 → 点 (960, 750) 领取 |

**陷阱**:
- 多个段位赛依次弹出(段位 + 战队 + 修罗)
- 每个都要点 (960, 750)
- 关掉一个会立刻弹出下一个

### 5.4 跨服要塞战倒计时 (中频)

| 项 | 内容 |
|---|---|
| 触发 | 跨服要塞战开始前 30 分钟 |
| 表现 | 屏幕中央"X 时 X 分 X 秒后开启" + 红字 |
| 位置 | 屏幕中央偏上 |
| 颜色 | 黄底红字 |
| 关闭方式 | **点 X 关闭** `(80, 60)` 或等倒计时 |
| 模板价值 | ⭐ 中(位置固定,颜色固定) |
| 自动化处理 | 检测到跨服要塞战倒计时 → 点 (80, 60) 关闭 |

### 5.5 战区聊天大字 (高频)

| 项 | 内容 |
|---|---|
| 触发 | 玩家在战区/世界/组织频道发言 |
| 表现 | 屏幕中央滚动大字 + 头像 + 文本 |
| 位置 | 屏幕中央(滚动) |
| 颜色 | 黄/白字 + 黑底 |
| 关闭方式 | **等几秒自动消失** |
| 模板价值 | ⭐ 低(内容是玩家输入,变) |
| 自动化处理 | **不用模板**(等自动消失) |
| 已知陷阱 | 临时遮挡中央按钮(可能挡"返回主页") |

### 5.6 网络异常弹窗 (中频)

| 项 | 内容 |
|---|---|
| 触发 | 服务器断开 / 网络超时 |
| 表现 | "网络连接断开" + "重连"按钮 |
| 位置 | 屏幕中央 |
| 颜色 | 白底黑字 |
| 关闭方式 | **"重连"按钮**(位置不固定) |
| 模板价值 | ⭐ 高(异常时必须处理) |
| 自动化处理 | OCR 兜底找"重连"按钮 → 点 → 等 → 验证 |

**变种**:
- "网络连接断开,请重试"
- "服务器连接失败"
- "登录超时,请重试"

### 5.7 更新提示 (低频)

| 项 | 内容 |
|---|---|
| 触发 | 游戏版本更新 |
| 表现 | "检测到新版本" + "立即更新"按钮 |
| 位置 | 屏幕中央 |
| 颜色 | 蓝底白字 |
| 关闭方式 | **"立即更新"** 或 **"取消"** 退出游戏 |
| 模板价值 | ⭐ 低(每月 1-2 次) |
| 自动化处理 | **需人工** — 强制更新无法跳过 |

### 5.8 维护公告 (低频)

| 项 | 内容 |
|---|---|
| 触发 | 服务器维护 |
| 表现 | "服务器维护中,预计 X 时恢复" |
| 位置 | 屏幕中央 |
| 颜色 | 红/灰底 |
| 关闭方式 | **无法关闭** |
| 模板价值 | ⭐ 低(变种多) |
| 自动化处理 | **任务 FAIL,需人工** |

### 5.9 登录公告 (中频)

| 项 | 内容 |
|---|---|
| 触发 | 登录后 |
| 表现 | "本周更新公告"全屏 + 滚动文本 + "我知道了"按钮(右下) |
| 位置 | 全屏 |

| 模板价值 | ⭐ 中(每周三维护后) |
| 自动化处理 | OCR 兜底找"我知道了" → 拖到底 → 点 |

### 5.10 充值礼包弹窗 (低频)

| 项 | 内容 |
|---|---|
| 触发 | 误点商店 / 活动推送 |

| 位置 | 屏幕中央 |

| 关闭方式 | **X 关闭** `(80, 60)` |
| 模板价值 | ⭐ 低(误触) |
| 自动化处理 | dismiss_x 关闭 |

## 7. 弹窗处理通用原则

### 7.1 dismiss_x 候选模板

- `shared/x.png` — 通用红 X(标准)
- `shared/x_white.png` — 白底 X
- `shared/x_left.png` — 左上角 X(主用)
- `shared/x_right.png` — 右上角 X(备用)
- `shared/close_btn.png` — 关闭按钮
- `shared/cancel_btn.png` — 取消按钮

> **关键**: X **永远在左上角**(2026-06-27 用户明确)。早期 dismiss_x 候选模板里有 `x_right.png`,实际是误判,优先级应该低于 `x_left.png`。

### 7.2 弹窗识别流程

```
[检测到非主页元素]
  ↓
[1. 扫 dismiss_x 候选模板(3 次)]
  ├─ 命中 → 点 X 关闭
  └─ 未命中 → 第 2 步
  ↓
[2. 扫"返回主页"按钮 (110, 990)]
  ├─ 命中 → 点返回
  └─ 未命中 → 第 3 步
  ↓
[3. 扫"确定"/"OK"/"我知道了"等通用按钮]
  ├─ 命中 → 点
  └─ 未命中 → 第 4 步
  ↓
[4. OCR 兜底]
  - 找"重连"/"确认"/"领取"等关键词
  ↓
[5. 强退(强制重启游戏)]
  - 3 次循环后仍未恢复
  - adb shell am force-stop
  - 重新打开游戏
```

### 7.3 已知陷阱

| 陷阱 | 原因 | 解决 |
|---|---|---|
| 教学剧情关不掉 | 点中央不会直接关,会"继续" | 多次点中央(2-3 次) |
| 段位奖励关完一个又来一个 | 段位 + 战队 + 修罗依次弹 | 多次扫 (960, 750) 领取 |
| 战区聊天长期不消失 | 玩家持续发言 | 等 30s 后强制关闭 |
| 跨服要塞战反复弹出 | 比赛前 30 分钟反复提醒 | dismiss_x 多次 |
| 公告需要拖到底 | "我知道了"按钮在右下 | 拖到底再点 |
| 充值弹窗无法关闭 | 强制充值活动 | dismiss_x 关闭 |
| 网络重连失败 | 服务器挂 | 5 次重试后 FAIL |
| 维护中无法恢复 | 服务器维护 | 任务 FAIL,需人工 |
| 更新提示无法跳过 | 强制更新 | 任务 FAIL,需人工 |
| 全屏活动 Banner 挡按钮 | 活动内容覆盖主页 | 先 dismiss_x 关闭 |
| **X 在左上角(不是右上角)** | 用户澄清 | dismiss_x 优先用 `x_left.png` |

## 8. 弹窗与任务的关联

| 任务 | 可能遇到的弹窗 | 处理 |
|---|---|---|
| mail | 邮件奖励弹窗 / 段位奖励 | 点"确定" / 段位"领取" |
| daily_signin | 签到成功弹窗 | 点中央 |
| weekly_signin | 签到成功弹窗 | 点中央 |
| liveness | 任务完成弹窗 | 点"确定" |
| group_signin | 教学剧情 / "即刻前往"弹窗 | 点中央 / 点"即刻前往" |
| recruit | 招募结果弹窗 | 点"确定" |
| activity | 活动弹窗 / 教学剧情 | dismiss_x / 点中央 |
