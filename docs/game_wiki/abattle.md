# 冒险 / 战斗 (Adventure / Battle)

> **战斗类任务的通用知识库**。所有涉及战斗的任务(剧情关卡/生存挑战/小队突袭/秘境探险/修罗副本/精英副本/...)的入口/出口/识别依据。

---

## 1. 任务名称

冒险副本(Adventure / Battle)及战斗类任务通用规则

## 2. 任务作用

- 进入战斗副本
- 完成战斗获得奖励(经验/铜币/装备/忍者碎片/...)
- 战斗失败的恢复

## 3. 入口位置

### 主页 → 冒险白底竖牌
```
[主页]
  ↓ 点击主页右下"冒险"白底竖牌图标 (1810, 1020)
[冒险副本页]
  ↓ 左侧副本分类 tab(剧情/精英/生存/...)
  ↓
  ↓ 选择具体副本(关卡)
  ↓
  ↓ 点击副本 → "开始战斗" / "立即前往"
[战斗开始]
```

### 主页 → 忍界指引 → 副本 tab
```
[主页]
  ↓ 点击主页右下"忍界指引"卷轴 (1580, 940)
[忍界指引页]
  ↓ 左侧菜单 swipe 下滑找副本 tab
  ├─ 剧情关卡
  ├─ 生存挑战
  ├─ 修罗副本
  ├─ 修罗天赋
  ├─ 团队副本
  ├─ 秘境探险
  └─ 忍者大赛
  ↓ 点击对应 tab
[副本页]
  ↓
  ↓ 选关 → 开始战斗
[战斗开始]
```

## 4. 页面截图说明

### 4.1 主页(冒险入口)

- **位置**: 右下角白底竖牌图标,有效 tap 坐标 `(1810, 1020)`
- **视觉中心**: `(1800, 950)`
- **颜色**: 白底 + 黑色"冒 险"字 + 红色装饰边框 + 绳子缠绕 + 底部黄色云朵
- **云朵装饰**: 是图标本体的一部分(2026-06-27 user 澄清)

### 4.2 冒险副本页

- **顶部**: "冒险"标题 + 关闭 X
- **左侧**: 副本分类 tab(剧情 / 精英 / 生存 / ...)
- **中部**: 副本关卡网格(每张副本卡片:封面 + 难度 + 推荐战力 + "开始"按钮)
- **底部**: 当日剩余体力 + 体力购买入口

### 4.3 战斗开始

- **加载画面**: 副本封面 + "Loading..."
- **进入战斗**: 全屏 3D 战斗场景

### 4.4 战斗中

- **顶部**: 玩家血条 + 敌人血条 + 倒计时
- **中部**: 战斗场景(3D 角色 + 技能特效)
- **底部**: 技能按钮(普攻 + 3 个技能 + 通灵 + 秘卷 + 替身)
- **左侧**: 暂停按钮 + 战斗信息

### 4.5 战斗结束(胜利)

- **中央**: 战斗结算
  - 经验值
  - 铜币
  - 装备
  - 忍者碎片(如有)
- **底部**: "再次挑战" / "退出" 按钮

### 4.6 战斗结束(失败)

- **中央**: "战斗失败" + "再试一次" / "退出" 按钮

## 7. 任务完整流程

### 7.1 战斗类任务通用流程

```
[主页]
  ↓ find_xxx_entry 节点(对应入口)
[副本页]
  ↓
  ↓ 选关 → find_stage_card 节点
[副本详情]
  ↓
  ↓ find_start_battle 节点
[战斗加载]
  ↓
[战斗开始]
  ├─ 等待战斗自动结束(3-5 分钟)
  └─ 或手动操作(技能 + 走位)
[战斗结束]
  ├─ 胜利 → find_victory_rewards 节点 → 找"退出"或"再次挑战"
  └─ 失败 → find_defeat_retry 节点 → 找"退出"或"再试一次"
[副本详情]
  ↓
  ↓ 关闭副本页
[主页]
  ↓
  ↓ verify_done 节点
[SUCCESS]
```

## 8. 成功条件

- **主页判定通过**
- **战斗已完成**(胜利/失败都算)
- **战斗奖励已结算**(已点"退出"或"再次挑战")
- **没有异常弹窗**

## 9. 失败情况

| 情况 | 原因 | 解决 |
|---|---|---|
| 体力不足 | 体力不够 | 买体力 或 任务 FAIL |
| 战斗超时 | 网络慢 | 重连 |
| 战斗中卡死 | 服务器卡 | 等 30s 后按"暂停" → 退出 |
| 战斗失败 | 操作不当 | 接受失败(降低难度重试) |
| 副本未解锁 | 等级不够 | 任务 FAIL,需人工 |
| 教学弹窗挡 | 第一次进副本 | dismiss_x 关闭 |
| 网络异常 | 战斗中断线 | 自动重连 → 进战斗结算 |

## 10. 恢复方式

### 10.1 战斗中恢复

```
[战斗中]
  ↓
[1. 按"暂停"按钮(左上)]
  ↓
[2. 选"退出战斗"或"投降"]
  ↓
[副本详情]
  ↓
[3. 关闭副本页]
  ↓
[主页]
```

### 10.2 战斗中断线恢复

```
[战斗中断线]
  ↓
[网络异常弹窗]
  ↓
[1. 点"重连"]
  ↓
[2. 重连成功 → 战斗继续]
  ↓
[3. 战斗结束 → 结算页]
  ↓
[4. 找"退出" → 主页]
```

### 10.3 战斗卡死恢复

```
[战斗中卡死(>30s 无响应)]
  ↓
[1. 按"暂停"按钮]
  ↓
[2. 选"退出战斗"]
  ↓
[副本详情]
  ↓
[3. 关闭副本页]
  ↓
[主页]
```

## 11. 自动化建议

### 11.1 战斗类任务识别依据(关键!)

| 状态 | 识别依据 |
|---|---|
| **开始战斗** | "战斗开始"提示 + Loading 画面 + 战斗场景 3D 渲染 |
| **战斗中** | 顶部血条 + 中部 3D 角色 + 底部技能按钮 + 倒计时 |
| **战斗结束(胜利)** | 中央"胜利" + 奖励列表 + "再次挑战"/"退出" |
| **战斗结束(失败)** | 中央"失败" + "再试一次"/"退出" |
| **战斗结算** | 经验/铜币/装备 列表 |
| **再次挑战** | "再次挑战"按钮(胜利时右侧) |
| **退出** | "退出"按钮(胜利/失败时右侧) |

### 11.2 推荐技术

| 场景 | 推荐技术 | 原因 |
|---|---|---|
| 主页冒险白底竖牌 | **多尺度模板匹配** + ROI + tap (1810, 1020) + tap_offset_y=-0.25 | 图标固定 |
| 冒险页 X | **多尺度模板匹配** + ROI | X 偏左上 |
| 副本卡片"开始"按钮 | **OCR 兜底** | 文字"开始" |
| **战斗加载画面** | **不用模板**(等自动) | 加载画面固定但短暂 |
| **战斗中 UI** | **不用模板** | 战斗中不需要识别(等自动) |
| 战斗胜利"再次挑战" | **OCR 兜底** | 文字"再次挑战" |
| 战斗胜利"退出" | **OCR 兜底** | 文字"退出" |
| 战斗失败"再试一次" | **OCR 兜底** | 文字"再试一次" |
| 战斗失败"退出" | **OCR 兜底** | 文字"退出" |
| 副本关卡封面 | **不用模板** | 章节变 |
| 战斗奖励 | **不用模板** | 副本变 |
| **战斗中断线** | **OCR 兜底** | 文字"重连" |
| **战斗中卡死** | **按"暂停"按钮**(左上) | 模板匹配 |
| **战斗类任务整体策略** | **等待战斗自动结束** | 不主动操作技能(等自动) |

### 11.3 战斗类任务特殊规则

1. **不要主动操作技能** — 等待战斗自动结束(3-5 分钟)
2. **战斗结束后点"退出"** — 不点"再次挑战"(避免循环)
3. **战斗中断线等待重连** — 不要主动退出
4. **战斗卡死按"暂停"退出** — 等 30s 无响应再操作
5. **体力不足不自动买体力** — 任务 FAIL,需人工
6. **战斗类任务允许失败** — 失败也返 SUCCESS(只要结算到主页)
7. **战斗 UI 不用识别** — 战斗中不需要识别(等自动结束)
