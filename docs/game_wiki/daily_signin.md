# 每日签到 (Daily Signin)

> 7 任务之一。**V1 已实现,真机回归待跑**(2026-06-27 dryrun_daily_signin 失败,模板不匹配)。

---

## 1. 任务名称

每日签到

## 2. 任务作用

- 每天领取签到奖励(铜币/钻石/体力/忍者碎片/...)
- 累计签到天数换大奖
- 每日 0 点刷新,漏签会断累计

## 3. 入口位置

**有 2 个入口**(需 user 确认哪个是真):

### 入口 A:主页 → 奖励中心 → 签到卡片
```
[主页]
  ↓ 点击右下"奖励"礼物盒 (1820, 555)
[奖励中心页]
  ↓ 点击"签到"卡片(左侧垂直 3 tab 第一项)
[签到页]
  ↓ 点击"签到"按钮(中央橙底大按钮)
[签到结果弹窗]
  ↓ 点击"确定"
[奖励中心页]
  ↓ 点击 X 关闭
[主页]
```

### 入口 B:主页 → 活动页 → 左侧下滑 → 每月签到
```
[主页]
  ↓ 点击右上"活动"图标 (1820, 85)
[活动页]
  ↓ 左侧菜单 swipe 下滑找"每月签到"
[点击"每月签到"]
  ↓
[签到页]
  ↓ 点击"签到"按钮
[签到结果弹窗]
  ↓
[活动页]
  ↓
[主页]
```

> **推荐用入口 A**(更直接,模板成熟);**入口 B 是 weekly_signin(月签)的实际路径**(详见 [activity.md](./activity.md))。

## 4. 页面截图说明

### 4.1 主页(奖励入口)

- **位置**: 右侧中部 `(~1820, 555)` 橙色礼物盒
- **颜色**: 橙底 + 金色蝴蝶结 + "奖励"白字 + 红点
- **检测**: 礼物盒轮廓 + 蝴蝶结 + "奖励"字

### 4.2 奖励中心页

- **顶部**: "奖励中心"标题 + 右上 X `(1860, 60)` 关闭
- **左侧**: 垂直 3 tab
  - 每日任务
  - 奖励中心(默认选中)
  - 日历
- **中部**: 卡片网格(签到 / 活跃 / 商城 / 限时 / ...)
- **每张卡片**: 标题 + 描述 + "立即前往" / "签到" 按钮
- **右侧**: 当日奖励进度 + 任务列表

### 4.3 签到页

- **顶部**: "签到"标题 + 累计天数 + 关闭 X
- **中部**: 7 天(或 30 天)日历网格
  - 已签:橙底 + 钩
  - 未签:灰底
  - 今日:亮底 + "签到"按钮
- **底部**: "签到"大按钮(橙底白字,`(1770, 920)` 附近)

### 4.4 签到结果弹窗

- **中央**: 奖励图标 + 数量
- **底部**: "确定"按钮

## 6. 页面变化元素

| 元素 | 变化原因 | 模板策略 |
|---|---|---|
| 签到累计天数 | 每天 +1 | 数字变,不能作为模板 |
| 7 天日历格子 | 每天变 | 不能作为模板 |
| 签到奖励内容 | 周期轮换 | 不能作为模板 |
| 奖励中心卡片 | 活动推送 | 多候选模板 |
| 顶部横幅 | 活动推送 | 不作为模板 |
| 背景主题 | 节日活动 | 不作为模板 |

## 7. 任务完整流程

### 入口 A(推荐)

```
[主页]
  ↓ find_award_button 节点(ROI (1760, 460, 200, 180), 礼物盒模板)
[检测到礼物盒 @ (1820, 555), conf=0.99]
  ↓ tap (1820, 555) + tap_offset_y=-37 → 实际 tap (1820, 518)
[奖励中心页]
  ↓
  ↓ find_signin_card 节点(找"签到"卡片)
[检测到"签到"卡片]
  ↓ tap 卡片中央
[签到页]
  ↓
  ↓ find_signin_button 节点(找"签到"按钮)
[检测到"签到"按钮 @ (1770, 920), conf=?]
  ↓ tap (1770, 920) + tap_offset_y=-0.25 → 实际 tap (1770, 895)
[签到结果弹窗]
  ↓
  ↓ 自动消失或点中央
[签到页(已签)]
  ↓
  ↓ 关闭签到页 → 关闭奖励中心
[主页]
  ↓
  ↓ verify_done 节点
[SUCCESS]
```

### 入口 B(月签路径,详见 [activity.md](./activity.md))

```
[主页] → 活动图标 → 活动页 → 左侧下滑 → "每月签到" → 签到页 → 签到按钮 → 主页
```

## 8. 成功条件

- **主页判定通过**
- **签到按钮状态变化**(已签/灰色)
- **签到结果弹窗显示奖励**
- **签到累计天数 +1**

## 9. 失败情况

| 情况 | 原因 | 解决 |
|---|---|---|
| 入口 A 礼物盒 conf<0.75 | 模板视觉差异大 | 多候选模板 + ROI 扩大 |
| 签到卡片找不到 | 卡片位置可能变 | OCR 兜底找"签到"文字 |
| 签到按钮被遮挡 | 弹窗阻挡 | dismiss_x 关闭后重试 |
| 今日已签 | 自动判断 | SUCCESS 退出 |
| 签到页 X 位置错 | 误以为在右上角 | 实际在 (1860, 60) 左上 |
| 签到页加载慢 | 服务器慢 | 等 1-2s 后重试 |
| 教学弹窗挡 | 第一次进签到页 | dismiss_x 关闭 |
| 段位奖励弹窗挡 | 段位赛结束 | dismiss_x 关闭 |

## 10. 恢复方式

```
[检测到非主页元素 / 任务卡住]
  ↓
[1. 关闭弹窗]
  - dismiss_x 候选模板扫描 3 次
  - 签到页 X (1860, 60)
  - 奖励中心 X (1860, 60)
  ↓
[2. 重试任务]
  - find_award_button → tap
  - find_signin_card → tap
  - find_signin_button → tap
  ↓
[3. 验证主页]
  - 检测资源栏 + 6 圆盘按钮
  ↓
[4. 主页 → SUCCESS]
  - 3 次循环后仍未到主页 → 报 FAIL
```
