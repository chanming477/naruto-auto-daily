## 🗂️ 文档结构

每个任务/页面文档统一 11 段:

1. **任务名称** — 简短描述
2. **任务作用** — 业务目标
3. **入口位置** — 完整路径(主页 → ... → 按钮)
4. **页面截图说明** — 描述页面长什么样
5. **页面固定元素** — 不变的:标题/图标/返回键/关闭按钮
6. **页面变化元素** — 经常变:Banner/广告/限时活动
7. **任务完整流程** — 流程图(主页 → A → B → C → 主页)
8. **成功条件** — 怎么算完成
9. **失败情况** — 哪些情况会失败
10. **恢复方式** — 出错后怎么回到主页/重新进入
11. **自动化建议** — 适合用什么技术(模板/OCR/固定点击/滑动)
