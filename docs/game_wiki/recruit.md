# 招募 (Recruit)

> 7 任务之一。**V1 已实现,真机回归待跑**(2026-06-27 dryrun_recruit 失败,模板不匹配)。

---

## 1. 任务名称

招募(忍者招募 / 高招 / 普招)

## 2. 任务作用

- 用铜币/钻石/招募券招募忍者
- 完成"免费招募 1 次"日常任务
- 累计招募换奖励

## 3. 入口位置

### 入口 A:主页 → 右上"招募"忍者图标
```
[主页]
  ↓ 点击右上"招募"忍者图标 (1810, 130) 附近
[招募页]
  ↓ 选择招募方式(高级招募 / 普通招募)
  ↓ 点击"免费招募" / "招募一次"
  ↓ 点击"招募"
[招募结果页]
  ↓ 显示招募到的忍者 + 分享/确认按钮
  ↓
  ↓ 关闭
[主页]
```

### 入口 B:主页 → 活动页 → 左侧菜单 → 招募
```
[主页]
  ↓ 点击右上"活动"图标 (1820, 85)
[活动页]
  ↓ 左侧菜单(可能含"招募" tab,需 swipe 找)
[招募 tab]
  ↓ 完成招募
  ↓
  ↓
[主页]
```

> **推荐用入口 A**(主页右上"招募"忍者图标)。

## 4. 页面截图说明

### 4.1 主页(招募入口)

- **位置**: 右上角垂直图标栏 `(~1810, 130)` 忍者立绘图标
- **颜色**: 主题色 + "招募"字
- **检测**: 忍者立绘 + "招募"字

### 4.2 招募页

- **顶部**: "招募"标题 + 关闭 X
- **中部**: 招募方式选择
  - 高级招募(钻石招募)
  - 普通招募(铜币招募)
  - 限时招募(活动)
- **底部**: 招募按钮区
  - "招募一次"(单次)
  - "招募十次"(十连)
  - "免费招募"(每 24h 1 次)

### 4.3 招募动画

- **中央**: 招募动画(忍术/苦无/树叶)
- **耗时**: 3-5s
- **自动进入结果页**

### 4.4 招募结果页

- **中央**: 招募到的忍者立绘 + 名字 + 评级(N/R/SR/SSR)
- **底部**: "分享" / "再招一次" / "确定" 按钮

## 8. 成功条件

- **主页判定通过**
- **招募成功(无论结果)**
- **"免费招募"已使用或"招募一次"已用铜币/钻石**
- **没有异常弹窗**

## 9. 失败情况

| 情况 | 原因 | 解决 |
|---|---|---|
| 招募入口 conf<0.75 | 模板视觉差异大 | 多候选模板 + ROI 扩大 |
| 没有免费招募 | 已用过 | 用铜币/钻石招募(消耗资源) |
| 铜币/钻石不足 | 资源不够 | **best-effort SUCCESS**(任务完成,只是没资源) |
| "招募一次"按钮找不到 | 招募页加载慢 | 等 1-2s 后重试 |
| 招募动画卡住 | 服务器慢 | 等 5s,超时后按 (110, 990) 返回 |
| 招募结果页"确定"找不到 | 模板不匹配 | OCR 兜底找"确定" |
| 教学弹窗挡 | 第一次进招募页 | dismiss_x 关闭 |
| 网络异常 | 招募时断线 | 重新进招募页 |

## 10. 恢复方式

```
[检测到非主页元素 / 任务卡住]
  ↓
[1. 关闭弹窗]
  - dismiss_x 候选模板扫描 3 次
  - 招募页 X (左上)
  - 招募结果页"确定"
  ↓
[2. 重试任务]
  - find_recruit_entry → tap
  - find_free_recruit → tap
  - find_confirm → tap
  ↓
[3. 验证主页]
  - 检测资源栏 + 6 圆盘按钮
  ↓
[4. 主页 → SUCCESS]
  - 3 次循环后仍未到主页 → 报 FAIL
```
