# 活动中心 (Activity)

> 7 任务之一(含 weekly_signin 每月签到)。**V1 已实现,真机回归待跑**(2026-06-27 dryrun_activity 部分通过,dryrun_weekly_signin 失败)。

---

## 1. 任务名称

活动中心(Activity)+ 每月签到(Weekly Signin)

## 2. 任务作用

### 活动中心
- 浏览所有限时活动
- 跳转到具体活动页(Banner → 活动详情 → 任务)
- **日常活动**入口:每月签到(weekly_signin)、节日活动、限时挑战

### 每月签到(weekly_signin)
- 每月累计签到 7/15/30 天换奖励
- 漏签会断累计
- 每月 1 号 0 点刷新

## 3. 入口位置

### 入口 A:主页 → 活动图标 → 活动页 → 左侧菜单 → 每月签到
```
[主页]
  ↓ 点击主页右上"活动"图标 (1820, 85)
[活动页]
  ↓ 左侧菜单 swipe 下滑找"每月签到"(深层菜单)
  ↓
  ↓ 点击"每月签到"
[签到页]
  ↓ 点击"签到"按钮(中央橙底大按钮)
[签到结果弹窗]
  ↓ 点击"确定"
[活动页]
  ↓ 点击 X 关闭
[主页]
```

> **这是 6/26 user 手动验证过的真实路径**(详见 `docs/game_wiki/weekly_signin.md` 历史版本)。

### 入口 B:主页 → 奖励中心 → 签到卡片(月签)
- 走 daily_signin 路径(见 [daily_signin.md](./daily_signin.md))
- 实际上是"每日签到",不是"每月签到"

## 4. 页面截图说明

### 4.1 主页(活动入口)

- **位置**: 右上角 `(1820, 85)` "活动"图标
- **颜色**: 主题色 + "活动"字
- **检测**: "活动"字 + 图标轮廓

### 4.2 活动页

- **顶部**: "活动"标题 + 关闭 X
- **左侧菜单**(可上下滑动):

- **中央**: 奖励图标 + 数量

## 8. 成功条件

### 活动中心
- 主页判定通过
- 活动页已打开并已关闭

### 每月签到
- 主页判定通过
- 签到按钮状态变化(已签/灰色)
- 签到累计天数 +1
- 签到结果弹窗显示奖励

## 9. 失败情况

| 情况 | 原因 | 解决 |
|---|---|---|
| 活动入口 conf<0.75 | 模板视觉差异大 | 多候选模板 + ROI 扩大 |
| 左侧菜单"每月签到"找不到 | 菜单可滑动 + 深层位置 | 多次 swipe 下滑 |
| 签到按钮被遮挡 | 弹窗阻挡 | dismiss_x 关闭后重试 |
| 今日已签 | 自动判断 | SUCCESS 退出 |
| 签到页 X 位置错 | 误以为在右上角 | 实际在左上 |
| 签到页加载慢 | 服务器慢 | 等 1-2s 后重试 |
| 教学弹窗挡 | 第一次进活动页 | dismiss_x 关闭 |
| 活动页 X 关闭后仍挡主页 | 弹窗没关 | 再点一次 X |
| 段位奖励弹窗挡 | 段位赛结束 | dismiss_x 关闭 |

## 10. 恢复方式

```
[检测到非主页元素 / 任务卡住]
  ↓
[1. 关闭弹窗]
  - dismiss_x 候选模板扫描 3 次
  - 签到页 X (左上)
  - 活动页 X (右上)
  ↓
[2. 重试任务]
  - find_activity_entry → tap
  - 左侧菜单 swipe 下滑
  - find_weekly_sign → tap
  - find_signin_button → tap
  ↓
[3. 验证主页]
  - 检测资源栏 + 6 圆盘按钮
  ↓
[4. 主页 → SUCCESS]
  - 3 次循环后仍未到主页 → 报 FAIL
```

## 11. 自动化建议

| 场景 | 推荐技术 | 原因 |
|---|---|---|
| 主页"活动"图标 | **多尺度模板匹配** + ROI (1750, 0, 200, 200) + tap (1820, 85) | 图标固定 |
| 活动页 X | **多尺度模板匹配** + ROI (1800, 0, 120, 120) | X 偏右上 |
| 左侧菜单 swipe | **连续 swipe 下滑 6 次** | "每月签到"在深层 |
| "每月签到" 菜单项 | **OCR 兜底** | 菜单项顺序可能变 |
| 签到按钮 | **多尺度模板匹配** + ROI + tap (1770, 920) + tap_offset_y=-0.25 | 按钮固定 |
| 签到结果弹窗"确定" | **dismiss_x 关闭** | 自动消失或点中央 |
| 签到页 X | **dismiss_x 关闭** | 实际在左上 |
| 活动 Banner | **不用模板** | 每周变 |
| 签到累计天数 | **不用数字** | 数字变 |
| 7 天 / 30 天日历 | **不用模板** | 每天变 |
